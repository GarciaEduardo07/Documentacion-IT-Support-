# Calculadora Mejorada (Python)

## Descripción
Esta es una calculadora en Python que permite:
- Sumar y multiplicar listas de números.
- Verificar si un número es par.
- Verificar si un número es entero.

Se usa desde la consola mediante un menú interactivo.

## Funciones

- `addmultiplenumbers(lista)` → Suma todos los números de una lista.
- `multiplymultiplenumbers(lista)` → Multiplica todos los números de una lista.
- `isiteven(numero)` → Devuelve True si el número es par e entero.
- `isitaninteger(numero)` → Devuelve True si el número es entero.

## Cómo usar

1. Ejecuta `main.py`:
2. Elige una opción del menú:
- 1: Sumar lista
- 2: Multiplicar lista
- 3: Revisar si un número es par
- 4: Revisar si un número es entero
- 5: Salir

3. Ingresa los números según la opción y ve el resultado.

## Requisitos
- Python 3.x
- No requiere librerías externas.
